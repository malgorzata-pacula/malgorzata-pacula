##  Introduction

I specialize in **high-quality manual data annotation** for AI systems, with a strong focus on **consistency, edge-case handling, and annotation reasoning**.

My work emphasizes:
- clean, machine-readable annotations  
- clear annotation guidelines  
- thoughtful handling of ambiguity  
- realistic human annotation workflows  

I use professional annotation tools and produce outputs suitable for **model training and evaluation**.
